== Description ==

Create youtube and other video links in your posts, and have them show up as links in facebook.

== Installation ==

1. Download youtube.php and place in wp-content/plugins
2. Log into your WordPress admin panel
3. Go to Plugins and "Activate" the plugin
4. Use youtube({videoid}); in your templates or in your posts by installing php-exec
	http://www.youtube.com/watch?v=JgHnqz4C-b8
	http://www.youtube.com/watch?v={videoid}
	youtube('JgHnqz4C-b8');
